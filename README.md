# Stream Discharge Forecasting in the Susquehanna River Basin

As part of my Senior Capstone graduation requirements at Washington College, I develop machine learning models to forecast stream discharge. The forecasting models used include Long Short Term Memory (variation of recurrent neural networks) and AutoRegressive Integrated Moving Average.

If you're looking for background, read through my submitted paper, Hudak_Thesis.

If you're interested in the code, look at main.py and functions.py. I had an easier time developing the code after separating it into two files. I appreciate any feedback on code quality and improvements. Feel free to fork, access the data portals, and plug your own data in. (The program took ~3 hours to run on a 4-core machine.)

Also, check out the results folder for the final output, which includes some quick pyplot graphs.
